## Installation and usage

## Installation
### Using yarn
```console
yarn add everpay
```

### Using yarn
```console
npm install everpay
```

## Usage

### es module
```js
import Everpay from 'everpay'
const everpay = new Everpay()
```

### commonjs
```js
const Everpay = require('everpay')
const everpay = new Everpay()
```
